# Essentials Starter Pack

## Skills to apply

Read and apply these skills sequentially or as needed:

- `.agent/skills/skills/concise-planning`
- `.agent/skills/skills/lint-and-validate`
- `.agent/skills/skills/systematic-debugging`
- `.agent/skills/skills/git-pushing`
- `.agent/skills/skills/kaizen`

## Execution rules

1. **Always start with a plan** using `concise-planning`.
2. **Implement the plan** and use `lint-and-validate` to ensure code quality.
3. If errors occur, switch to `systematic-debugging`.
4. When done, use `git-pushing` to commit the work.
5. End with `kaizen` for continuous improvement.
